## Key points to cover
- Explain the root cause precisely: ChatGPT's interface renders Markdown visually, but copying from the rendered chat window pastes inconsistent rich-text/HTML rather than the underlying Markdown — which is why formatting comes through wrong or inconsistently across browsers.
- Give the exact workaround: use ChatGPT's "copy" button (which copies raw Markdown source on most clients) rather than manually selecting and copying rendered text, then run that raw Markdown through a converter.
- Keep the use case grounded: people doing this are usually turning an AI-drafted report, summary, or outline into a document to share with someone who doesn't use ChatGPT — frame the workflow around that, not generic "AI content" hype.
